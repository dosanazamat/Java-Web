# Промежуточная аттестация — Web & Java (тестовая игра)

Тестовый тренажёр по двум предметам (**Web** и **Java**) на основе экзаменационного материала AITU College, 2025–2026.

- **9 вариантов × 40 вопросов** в каждом предмете (всего 720 вопросов).
- Порядок вопросов и вариантов **строго совпадает** с Word-документами.
- На каждый вопрос — **30 секунд**.
- После ответа сразу видно: верно/нет и какой ответ правильный.
- В конце — статистика, оценка («Отлично / Хорошо / Удовлетворительно / Плохо») и разбор всех ответов.

## Структура папки

```
quiz-game-final/
├─ index.html       # точка входа
├─ styles.css       # стили
├─ app.js           # вся логика игры
├─ data.js          # вопросы + ответы (встроенные)
├─ data/
│  ├─ web.json      # Web — вопросы и ответы (JSON, для редактирования)
│  └─ java.json     # Java — то же самое
└─ README.md
```

## Как запустить локально

### Вариант 1. Двойной клик
Просто открой `index.html` в браузере. Должно сразу заработать — все данные встроены в `data.js`, никакого сервера не требуется.

### Вариант 2. Локальный сервер (рекомендую для разработки)

```bash
# Из папки quiz-game-final/
python3 -m http.server 8000
# открой http://localhost:8000
```

или через Node:
```bash
npx serve .
```

или плагин **Live Server** в VS Code.

## Как задеплоить

Это статический сайт — заливается куда угодно:

- **Vercel** / **Netlify**: перетащи папку `quiz-game-final` в дашборд → готово.
- **GitHub Pages**: запушь содержимое папки в репо, включи Pages в настройках.
- **Cloudflare Pages**: подключи репо или загрузи zip.
- **Любой хостинг**: загрузи файлы по FTP в публичную папку.

Бэкенд не нужен. Никаких внешних API не дёргаем.

## Если нашёл ошибку в правильном ответе

Ответы в Word-документах **не были помечены** — я их проставил по смыслу (по моим знаниям предметов). Где-то могут быть неточности.

**Чтобы исправить ответ:**

1. Открой `data/web.json` или `data/java.json`.
2. Найди нужный вариант и вопрос (поле `"n"` — номер вопроса).
3. Поменяй значение поля `"correct"` на правильную букву (`"A"`, `"B"`, `"C"`, `"D"` или `"E"`).
4. Перегенерируй `data.js`:

   ```bash
   # Если у тебя есть Python:
   python3 -c "
   import json
   with open('data/web.json', encoding='utf-8') as f: web = json.load(f)
   with open('data/java.json', encoding='utf-8') as f: java = json.load(f)
   js = 'window.QUIZ_DATA = {\n  Web: ' + json.dumps(web, ensure_ascii=False) + ',\n  Java: ' + json.dumps(java, ensure_ascii=False) + '\n};\n'
   open('data.js', 'w', encoding='utf-8').write(js)
   "
   ```

   Или просто открой `data.js` в редакторе и поправь нужный вопрос вручную (он там в виде одного большого объекта — найди по тексту вопроса через Ctrl+F).

## Технологии

Чистый HTML + CSS + JS, без фреймворков, без сборки. Шрифты — Google Fonts (`Unbounded`, `Manrope`, `JetBrains Mono`).

## Как менять время на вопрос

В `app.js`, в самом верху:

```js
const CONFIG = {
  SECONDS_PER_QUESTION: 30,  // ← поменяй здесь
};
```
